# Лабораторна 1 — Задача 2

## Умова
Програма зчитує два дійсні числа та обчислює їх середнє арифметичне.

## Алгоритм
1. Користувач вводить два числа `a` і `b`.
2. Програма обчислює `(a + b) / 2`.
3. Результат виводиться на екран.

## Використані оператори C++
- `cin` — введення даних.
- `cout` — виведення результату.
- `<<`, `>>` — оператори потоків.
- `return 0;` — завершення програми.

## Приклад роботи
Введіть два числа: 4 10
Середнє арифметичне: 7

## Інструкція запуску
1. Відкрити проєкт `lab1_task2.sln` у Visual Studio.  
2. Натиснути **Ctrl + F5**.  
3. Увести два числа.
