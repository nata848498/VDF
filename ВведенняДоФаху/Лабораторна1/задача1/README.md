# Лабораторна 1 — Задача 1

## Умова
Програма зчитує два цілі числа і виводить їхню суму.

## Алгоритм
1. Користувач вводить два числа `a` і `b`.
2. Програма обчислює `a + b`.
3. Результат виводиться на екран.

## Використані оператори C++
- `cin` — введення даних.
- `cout` — виведення результату.
- `<<`, `>>` — оператори потоків.
- `return 0;` — завершення програми.

## Приклад роботи
Введіть два числа: 3 7
Сума: 10

## Інструкція запуску
1. Відкрити проєкт `lab1_task1.sln` у Visual Studio.  
2. Натиснути **Ctrl + F5**.  
3. Увести два числа.  
4. Переглянути результат у консолі.# ‡ ¤ з  1
