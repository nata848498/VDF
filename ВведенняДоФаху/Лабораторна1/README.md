# Лабораторна робота №1

## Тема
Ознайомлення з основами програмування мовою C++.

## Мета
Навчитися створювати прості консольні програми, виконувати введення та виведення даних, а також застосовувати базові оператори мови C++.

## Список задач
1. **Задача 1** — обчислення суми двох чисел.  
2. **Задача 2** — знаходження середнього арифметичного двох чисел.

## Алгоритм виконання
1. Для кожної задачі створено окрему папку (`задача1`, `задача2`).  
2. У кожній міститься:
   - файл з кодом `.cpp`;
   - файл `README.md` з описом алгоритму, прикладом та інструкцією запуску.  
3. Кожну задачу запущено у Visual Studio (Ctrl + F5) та перевірено.  
4. Усі результати додано до GitHub.

## Висновок
Під час виконання лабораторної роботи я ознайомилася з базовим синтаксисом C++, навчилася створювати проєкти у Visual Studio, працювати з операторами введення/виведення (`cin`, `cout`) і використовувати GitHub для збереження результатів.
